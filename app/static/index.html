Scénario conseillé pour la soutenance

Tu peux suivre ces étapes devant le professeur pour montrer les méthodes REST et les relations.

  1. Créer un auteur dans l’onglet Auteurs.
  2. Créer un livre lié à cet auteur dans l’onglet Livres.
  3. Créer un lecteur dans l’onglet Lecteurs.
  4. Créer une carte pour ce lecteur dans l’onglet Cartes.
  5. Créer un emprunt dans l’onglet Emprunts.
  6. Retourner le livre avec le bouton Retourner.
  7. Ouvrir Swagger pour montrer l’API REST complète.
